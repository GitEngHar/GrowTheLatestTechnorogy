### GLTサイクルモデル
<details>
<summary> 詳細 </summary>
 ■ GLTサイクルモデルってなに？
継続的にコンテンツを作りプロジェクトを拡大していく取り組み

■ 何するの？
デモ作成 → レビュー会・コンテンツ概要設計 → コンテンツ作成 → コンテンツレビュー →改善
コンテンツ概要設計では、コンテンツに何が必要なのか、コンテンツはどんな構造になるのかを大雑把に作成致します。

■ どれくらいの頻度？
まずは１サイクル1か月のスパンで回していく✨
